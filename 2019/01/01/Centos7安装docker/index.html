Centos7安装docker
发布于 2019-01-01 / 更新于 2019-01-08 / Linux docker / 热度 0 / 喜欢 0

卸载旧版本dockek

sudo yum remove docker \
                  docker-client \
                  docker-client-latest \
                  docker-common \
                  docker-latest \
                  docker-latest-logrotate \
                  docker-logrotate \
                  docker-selinux \
                  docker-engine-selinux \
                  docker-engine

安装所需要的包，yum-utils提供了yum-config-manager 效用，并device-mapper-persistent-data和lvm2由需要devicemapper存储驱动程序。

sudo yum install -y yum-utils \
  device-mapper-persistent-data \
  lvm2

设置稳定存储库

sudo yum-config-manager \
    --add-repo \
    http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

安装最新的docker-ce

sudo yum install docker-ce

查看仓库中可用的docker版本

yum list docker-ce --showduplicates | sort -r

安装特定版本

sudo yum install docker-ce-

添加镜像源/etc/docker/daemon.json

{
  "registry-mirrors": ["https://registry.dockercn.com","https://ko1ow3tj.mirror.aliyuncs.com"]
}

启动docker

sudo systemctl start docker
